🩺 Breast Cancer Detection using Machine Learning

📌 Project Overview

This project predicts whether a breast tumor is Benign or Malignant using Machine Learning techniques. The model is trained on breast cancer diagnostic data and helps demonstrate how AI can assist in healthcare-related predictions.

🎯 Objective

To build a classification model that can accurately detect breast cancer based on medical features and improve understanding of machine learning applications in healthcare.
